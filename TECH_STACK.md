# Tech Stack (Web 2D PoC)

## Required
- TypeScript (strict)
- Vite
- HTML5 Canvas 2D (single canvas)
- Minimal DOM overlay for UI (Restart button + win/lose text)

## Not allowed (PoC)
- React/Vue/Angular
- Pixi/Phaser/Three/Babylon
- Any server/backend
- Any external state management library

## Runtime targets
- Desktop Chrome/Edge/Firefox latest
- 60 FPS goal on typical laptop

## Build & run
- `npm install`
- `npm run dev`
- `npm run build`
